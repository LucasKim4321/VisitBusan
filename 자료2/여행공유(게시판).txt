- **글쓰기:** 불가 (로그인 안내 문구가 뜸).

- **공유:** 게시글을 카톡, 인스타, 페이스북 등으로 공유 가능.
- **신고:** 불가 (로그인 안내 문구가 뜸).
- **글 수정 및 삭제:** 불가.
- **검색 및 태그:** 카테고리, 기간, 검색기준, 검색어를 설정하여 게시글 검색 가능. 인기 태그 선택 시 해당 태그를 가진 글만 표시.
- **페이지 넘기기:** 게시글 페이지를 넘기며 탐색 가능.

📌 **회원 유저**

**권한 및 기능**

- **게시글 보기:** 모든 게시글과 공지사항을 볼 수 있음.
- **글쓰기:** 로그인한 회원만 글쓰기 가능. 게시글 작성 시 제목, 장소링크, 글내용(사진, 동영상, 글 등), 태그 입력 가능.
- **좋아요:** 다른 유저가 쓴 글에 좋아요 체크 on/off 가능. 마이페이지에서 좋아요 누른 게시글 확인 가능.
- **댓글:** 댓글 작성 가능. 자신이 쓴 댓글 수정 및 삭제 가능.
- **공유:** 게시글을 카톡, 인스타, 페이스북 등으로 공유 가능.
- **신고:** 신고하기 버튼 누르면 신고 카테고리 선택 후 신고 가능.
- **글 수정 및 삭제:** 자신이 작성한 글은 수정 및 삭제 가능.
- **검색 및 태그:** 카테고리, 기간, 검색기준, 검색어를 설정하여 게시글 검색 가능. 인기 태그 선택 시 해당 태그를 가진 글만 표시.
- **페이지 넘기기:** 게시글 페이지를 넘기며 탐색 가능.
</aside>

📌 **관리자**

**권한 및 기능**

- **회원 유저 권한:** 모든 회원 유저 기능 포함.
- **공지사항 작성:** 게시판 최상단에 공지사항 작성 가능. 모든 게시글과 동일한 형식으로 등록, 수정, 삭제 가능.
- **게시글 삭제:** 모든 유저의 게시글 삭제 가능.
- **신고 접수 및 처리:** 신고가 접수된 글 확인 후 삭제, 제제, 문제없음 처리 가능. 신고 태그 활성화. 신고 처리 후 신고 태그 자동 삭제.
- **신고 관리:** 신고 접수된 글은 관리자 마이페이지에서 확인 가능. 신고된 유저의 신고당한 글, 처리상태, 신고당한 횟수 확인 가능.
- **페이지 넘기기:** 게시글 페이지를 넘기며 탐색 가능.
- **검색 및 태그:** 카테고리, 기간, 검색기준, 검색어를 설정하여 게시글 검색 가능. 인기 태그 선택 시 해당 태그를 가진 글만 표시.
</aside>

<aside>
📌 비회원 유저

**기능 명세화**

**User Story 1: 게시글 작성하기**

- **사용자 유형:** 회원 유저
- **목표:** 새로운 여행 후기를 게시판에 작성하고 공유한다.
- **시나리오:**
    1. 사용자는 로그인 후 여행 공유 게시판에 접속한다.
    2. 상단의 글쓰기 버튼을 클릭한다.
    3. 글쓰기 화면에서 카테고리, 제목, 장소링크, 사진 첨부, 글 내용을 입력하고 태그를 추가한다.
    4. 등록 버튼을 클릭하여 게시글을 게시한다.
    5. 작성된 게시글이 게시판에 표시되고 다른 유저들이 볼 수 있게 된다.
    

**User Story 2: 좋아요 및 댓글 달기**

- **사용자 유형:** 회원 유저
- **목표:** 다른 유저의 게시글에 좋아요를 누르고 댓글을 남긴다.
- **시나리오:**
    1. 사용자는 게시판에서 관심 있는 게시글을 선택하여 상세 내용을 본다.
    2. 게시글 하단의 좋아요 버튼을 클릭하여 좋아요를 누른다.
    3. 댓글창에 댓글을 작성하고 등록 버튼을 클릭한다.
    4. 좋아요와 댓글이 반영되며, 게시글 작성자는 이를 확인할 수 있다.
    

**User Story 3: 게시글 수정 및 삭제**

- **사용자 유형:** 회원 유저
- **목표:** 자신이 작성한 게시글을 수정하거나 삭제한다.
- **시나리오:**
    1. 사용자는 자신의 게시글을 선택하여 상세 페이지로 이동한다.
    2. 수정 버튼을 클릭하여 글쓰기 화면으로 이동하고, 내용을 수정한 후 수정하기 버튼을 클릭한다.
    3. 수정된 내용이 반영된 게시글이 표시된다.
    4. 삭제 버튼을 클릭하여 삭제 여부를 확인한 후 게시글을 삭제한다.
</aside>

    3. 게시글을 클릭하여 상세 내용을 읽는다.
    4. 필요한 정보를 얻고 페이지를 넘기며 다른 게시글도 탐색한다.

📌 관리자

**기능 명세화**

**User Story 1: 공지사항 작성 및 관리**

- **사용자 유형:** 관리자
- **목표:** 게시판 상단에 공지사항을 작성하고 관리한다.
- **시나리오:**
    1. 관리자는 로그인 후 게시판에 접속한다.
    2. 글쓰기 버튼을 클릭하여 공지사항 체크박스를 선택한다.
    3. 제목, 내용 등을 입력하고 등록 버튼을 클릭하여 공지사항을 게시한다.
    4. 필요시 공지사항을 수정하거나 삭제한다.

**User Story 2: 신고 접수 및 처리**

- **사용자 유형:** 관리자
- **목표:** 유저가 신고한 게시글을 확인하고 적절히 처리한다.
- **시나리오:**
    1. 관리자는 신고 접수 알람을 확인하고 마이페이지의 신고 접수 창에 접속한다.
    2. 신고된 게시글을 검토한 후 삭제, 제제, 문제없음 처리 중 하나를 선택하여 처리한다.
    3. 처리된 게시글은 신고 태그가 삭제되고, 유저의 신고당한 횟수가 기록된다.
</aside>
