<aside>
📌 **요구사항**

**고객**

- 일반회원이 게시글 등록 시 나의 리뷰에서 확인이 가능합니다.
- 일반회원은 여행공유 페이지에서 게시물을 찜할 수 있어야 한다.
- 찜한 게시물은 '찜한 여행지' 페이지에서 확인할 수 있어야 한다.
    
    (“부산에 가면 > 명소, 음식 / 추천여행 > 일정여행, 테마여행, 미식투어 의 좋아요 기능 경우는 로그인을 하지 않아도 가능)
    
- 회원정보 수정(아이디, 이름, 생년월일, 성별, 전화번호 변경 불가 비밀번호 및 프로필 수정)
- 회원 탈퇴 (비밀번호 입력 후 확인 버튼 누를 시 회원 탈퇴 및 회원 정보 모두 삭제)
- 프로필 사진 업로드 기능
- 프로필에 들어갈 간단한 소개글 작성 기능(30자 이내)
</aside>

<aside>
📌 **승인 기준**

**고객**

- 일반회원으로 로그인 후 게시글을 등록한 후, '나의 리뷰' 페이지에서 등록한 게시글이 정상적으로 표시되는지 확인합니다.
- 일반회원으로 로그인한 상태에서 여행공유 페이지에서 게시물에 찜하기 기능이 정상 작동하는지 확인합니다.
- 찜한 게시물이 '찜한 여행지' 페이지에 올바르게 표시되는지 확인합니다.
- 찜한 게시물이 '찜한 여행지' 페이지에 리스트업 되는지 확인합니다.
- 각 찜한 게시물이 해당 페이지에서 정확한 정보를 보여주는지 확인합니다.
- 회원정보 수정 페이지에서 아이디, 이름, 생년월일, 성별, 전화번호는 수정 불가능함을 확인합니다.
- 비밀번호 및 프로필 정보(프로필 사진, 소개글)는 정상적으로 수정 가능한지 확인합니다.
- 비밀번호 입력 후 탈퇴 버튼을 누르면 회원 탈퇴가 정상적으로 이루어지는지 확인합니다.
- 탈퇴 후 해당 회원의 모든 정보가 시스템에서 삭제되는지 확인합니다.
- 프로필 사진 업로드 기능이 정상적으로 작동하는지 확인합니다.
- 업로드한 사진이 프로필 페이지에 올바르게 표시되는지 확인합니다.
- 소개글 작성 기능이 정상 작동하는지 확인합니다.
- 30자 이내로 작성 가능하며, 작성한 소개글이 프로필 페이지에 올바르게 표시되는지 확인합니다.
</aside>
