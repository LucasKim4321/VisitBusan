# 통합관리시스템

세부기능 요구사항

- **사이트 관리**
    - 사이트 등록/수정/삭제 및 사용중지/모니터링 및 관리 기능
    - 사이트 별 운영 관리자 지정 및 접근 권한 지정 기능
    - 메인/서브 템플릿 관리기능
    - 원클릭 사이트 생성 기능(선택된 템플릿을 기반으로 기본설정)
    - 기존 운영 중인 사이트를 활용한 사이트 생성 기능
    - 홈페이지 생성 제한이 없는 라이센스 제공
    - 리소스 목록, 관리자정보, 백업/복구, 홈페이지 상태 변경, 템플릿 등록신청 관리
    
- **사용자 관리**
    - 홈페이지 전체 회원 목록 및 상세정보 관리
    - 사용자 그룹 생성/수정/삭제 및 회원 그룹관리 기능
    - 휴대폰 본인 실명 확인서비스 지원
    - 내부 조직도 연계를 통한 사용자 관리 기능
- **권한 관리**
    - 홈페이지, 콘텐츠, 게시판, 업무프로그램 등 권한을 세분화하고 권한별 시스템 메뉴 제한 기능
    - 권한은 유형별로 분할 하여 설정 기능
    - 권한설정에 따른 로그(계정 생성/수정/말소)를 기록
    - 발주자 시스템의 SSO와 연동하여 권한설정 부여 가능
- **메뉴 관리**
    - 홈페이지 메뉴 구조를 데이터베이스 기반으로 관리 가능
    - 홈페이지별 메뉴 등록/수정/삭제 및 출력순서 변경/이동 기능 (메뉴 4Depth 이상 지원)
    - 메뉴 표시(공개, 비공개)에 대한 설정 기능
    - 메뉴별 담당 부서(또는 담당자) 지정 및 노출 기능
    - 메뉴별 최종 수정일 출력(콘텐츠 수정 시 자동변경)
- **콘텐츠 관리**
    - 메뉴별 콘텐츠를 웹상에서 바로 수정, 삭제, 미리 보기 가능한 Live 수정 기능
    - 반응형 웹페이지의 UA(User Agent)별 화면 미리보기 기능
    - 콘텐츠 수정 시 히스토리 관리 및 백업, 복구 기능 제공
    - 반응형 콘텐츠 관리를 위한 HTML5 기반(Not Active-X) 에디터 기능
    - 제목, 문단, 웹 프로그래밍, 이미지 추가 및 삭제, 웹 표준이 준수된 자동마크업, Drag & Drop으로 요소의 위치 및 크기 변경, 콘텐츠 복사/붙여넣기 기능 제공
    - 통합 홈페이지의 모든 콘텐츠를 메타데이터로 검색에 활용할 수 있도록 구성
    - 콘텐츠 등록시 웹표준, 웹접근성, 웹호환성 준수할 수 있도록 방안제시
    - 콘텐츠 관리를 위한 콘텐츠 게시 승인 프로세스 시스템 기능 제공
- **로그관리**
    - 사이트별, 메뉴별 개별 접속 로그, SNS 유입정보의 기간별 로그 제공
    - 홈페이지 접속정보 중 기기/운영체제/브라우저/접속언어의 기간별, 메뉴별 통계정보를 제공
    - 게시판별 게시물 등록, 댓글 및 좋아요 로그 제공
    - 관리자 접속 및 접속 해지 로그 제공
- **배너/ 팝업 및 이미지 관리**
    - 배너/팝업 관리 : 데이터베이스 연계한 배너/팝업 관리기능 구현
    - 배너/팝업의 게시일자 예약 및 기간 설정, 순서변경 등 기능 제공
    - 대표 홈페이지의 경우 멀티 레이어 팝업 기능 구현 또는 팝업 전체 보기 기능 제공
    - 학과/기관 사이트의 경우 부서 관리자가 쉽게 등록할 수 있는 텍스트 팝업/배너 기능 구현(다양한 배경 스킨 제공)
    - 메인 이미지 : 데이터베이스 연계한 비주얼 관리기능
    - 이미지 게시기간 설정, 순서변경, 특정 페이지 링크 기능 등
    - 등록된 아이콘, 이미지 등만 사용 가능하도록 제어할 수 있는 이미지 라이브러리 관리기능
- **운영관리**
    - 단위 프로그램은 컴포넌트 형태로 개발하여 별도 커스터마이징 없이 추가 설치가 가능하도록 구성
    - 관계 법령에 따른 회원정보 보유기간, 비밀번호 초기화, 변경 주기 관리, 휴면처리 등 기능 제공
    - 관리자 페이지는 외부 인터넷에서 관리자 접속 불가능
    - 시스템 관리자 페이지는 특정 IP나 MAC 값으로만 접속
    - 로그인 및 비밀번호 등록 시 SSL 등 소통자료 암호화 처리
    - 지정한 횟수 로그인 실패 시 접속 중단 처리
    - 관리자 접근 권한 부여, 변경, 말소내용 기록 및 관리
    - 대량의 트래픽 발생시 분기페이지 등록/관리 기능
    - 홈페이지 제공 기본 용량, 언어, 콘텐츠별 변경 히스토리 수량 관리
    - 파일 업로드를 위한 가능/불가능 확장자, 파일 사이즈 관리
    - 비허용 단어 필터링 설정 및 불건전게시물 관리
