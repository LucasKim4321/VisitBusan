# 통합게시판

게시판을 쉽게 생성/삭제/이관/관리, 게시판별로 운영자를 추가/수정/삭제 구성
○ 필수 입력정보의 설정, 공개여부, 업로드 파일용량 제한, 첨부파일을 이용한 악성코드 삽입 제한(정해진 데이터 형식만 파일 등록)
○ 게시판 내용에 주민등록번호 등의 개인정보 입력 방지
○ 멀티게시판 관리 기능을 통한 다양한 유형(텍스트, 이미지, 동영상) 게시판 관리기능을 구현하여 다양한 형태의 자료를 게재할 수 있도록 기능 구현- 게시판의 추가, 삭제 등 추후 관리자가 원하는 형태의 게시판 생성 기능(자유형, 자료형, 비공개형, 갤러리, 보도자료형 등)
○ 게시물 SNS 공유, RSS 자동생성, 스크랩, 인쇄 기능 구현
○ 주요 게시물, 공지사항 등에 대한 SMS/E-mail/SNS 발송 기능 생성
○ 웹에디터 기능 제공, 게시물간 이동 기능 적용
○ 게시물에 답변이 있을 경우 질문과 답변을 동일한 화면에 표시하고 글 작성시 HTML, TEXT 방식을 지원
○ 별도의 뷰어 설치 없이 웹 페이지를 통해 첨부문서를 바로 확인 할 수 있는 문서뷰어 기능 제공
○ 다양한 파일(대용량 파일 포함)을 쉽게 업로드 할 수 있도록 구현- HWP, PDF, 동영상, 이미지 파일을 HTML5형태로 변환하여 브라우저 상에서 열람 기능- 호환성이 준수된 Non Active-X방식으로 멀티파일 업로드 기능- 첨부파일 업로드 시 드래그앤드롭 기능
○ 상담성 게시판은 접수, 처리중, 완료 등 처리 상태 표시 기능
○ 하나의 게시글을 복수개의 게시판을 선택 또는 전체 게시할 수 있도록 기능 구현
○ 썸네일 추출 시 이미지 왜곡현상이 없도록 구현
○ 구성원이 글 작성 시 자신의 소속 부서명으로 작성하는 기능 제공
○ 개인별, 구성원 그룹별 게시판 메뉴가 다르게 표현되는 설정 구성
○ 개인별, 구성원 그룹별 읽기/쓰기/삭제/답변 권한의 설정이 가능하도록 구성
○ 표준화, 확장성, 재사용성을 고려한 시스템 관리체계 구축
○ 게시글 중 선택적으로 공개/비공개로 설정이 가능하도록 구성
○ 필드의 추가/삭제 등 관리자가 원하는 형태의 게시판 생성 가능
○ 회원용/비회원용 및 비로그인 게시판 구축
○ 게시판 구성과 기능은 기존 홈페이지를 기준으로 학교의 요구사항을 수용하여 적용
○ 게시글 작성 시 데이터보호를 위한 임시저장 기능
○ 일부 게시판은 내부 학사 DB와 연동
